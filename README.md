# ⚡ ABC전기 전기제어 설계 자동화 Agent

> **실제 현장 업무에서 사용 중인 자동화 도구입니다.**

전기제어 설계팀이 매 프로젝트마다 수작업으로 작성하던 전기 용량표를 자동화한 웹 기반 설계 보조 도구입니다.  
**300개 이상의 부품 PDF에서 스펙을 자동으로 읽어와, 현재 부하에 맞는 최적 제품을 자동으로 선정해주는 역할**을 합니다.  
설치 없이 브라우저만으로 동작하며, AI 연동을 통한 설계 검토 기능을 제공합니다.

---

## 🔧 주요 기능

- **부하계산 자동화** — W, A(Load), A(차단기), 상전류(R/S/T/N) 자동 계산
- **제품 자동선정** — 300개+ 부품 PDF 파싱 DB 기반으로 CP · MCCB · SMPS 자동 선정
- **상 불평등률 분석** — 10% / 30% 기준 자동 경고
- **xlsx 다운로드** — 원본 포맷 그대로 Excel 출력
- **AI 설계 검토** — OpenAI API 연동, 현재 용량표 데이터 기반 질의응답
- **프로젝트 백업/복원** — JSON 파일로 저장 (파일명 날짜 자동 기입)
- **GPS / UPS 섹션 분리** — 전압 · Phase 자유 설정

## 🚀 사용법

### 접속
브라우저에서 아래 링크 접속 (설치 불필요)
```
https://smw312500-commit.github.io/Choi-AI-AGENT
```

### 기본 흐름
1. **프로젝트명 입력** — 상단 입력창에 장비명 입력
2. **GPS / UPS 설정** — 전압, Phase, 메인 차단기 정보 입력
3. **서브패널 추가** — MC1st / MC2nd 등 패널 구성
4. **부하 추가** — `+ 부하 추가` 버튼 → 부하 정보 입력 → 자동 계산 및 제품 선정
5. **용량표 확인** — 우측 `📊 용량표` 탭
6. **xlsx 다운로드** — 상단 `📥 xlsx 다운로드` 버튼
7. **저장** — `💾 저장` (로컬스토리지) 또는 `💾 JSON 백업` (파일)

### AI 채팅
1. `🤖 AI 채팅` 탭 이동
2. OpenAI API Key 입력 후 저장
3. 현재 용량표 기반으로 설계 질문 가능

### PDF 자동 파싱 (제품 DB 구축)
```
python parse_pdfs.py
```
→ API Key 입력 → PDF 폴더 경로 입력 → 자동 실행  
→ 생성된 `products_db.json`을 앱에서 불러오기
